# ICT Quarterly Theory

Source: Jevaunie Daye via time-price-research-astrofin.blogspot.com

## Core Concept

**Time must be divided into quarters for proper interpretation of market cycles.**

This is fractal - applies to yearly, monthly, weekly, daily, and session timeframes.

---

## Weekly Cycle Structure

**Four quarters, each one day long (Monday-Thursday)**

| Quarter | Day | Notes |
|---------|-----|-------|
| Q1 | Monday | Sets the tone, dictates following quarters |
| Q2 | Tuesday | Contains the "True Open" |
| Q3 | Wednesday | |
| Q4 | Thursday | |

**Friday is NOT included** - has its own specific function (profit-taking, reversal, retracement)

### Weekly True Open
- **18:00 EST every Monday**
- Trading strategy: Buy below true open when bullish, sell above when bearish

### Q1 Function
- Q1 dictates the quarters that follow
- Acts as a barometer

| Quarter | Time (EST) | Session |
|---------|------------|---------|
| Q1 | 18:00 - 00:00 | Asia |
| Q2 | 00:00 - 06:00 | London |
| Q3 | 06:00 - 12:00 | NY AM |
| Q4 | 12:00 - 18:00 | NY PM |

### Daily True Open
- **Midnight EST (00:00)**

## Trading Session Strategies

Based on how quarters unfold:

1. **Asia Range, London Manipulation, NY Reversal**
2. **Asia Expansion, London Consolidation, NY Continuation**
3. **Asia Range, London Expansion, NY Consolidation**
